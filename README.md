<!-- GETTING STARTED -->
## Getting Started


### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/edgeboy47/dexterhut_pm_backend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API in `.env`
   ```sh
   PORT = 3000
   ```
